This package will install Thonny VERSION to /Applications. If you already have another Thonny version in this location, it will be replaced.

You will be asked for admin password. If you don't have it or just you want to install Thonny elsewhere, then use thonny-VERSION-mac-portable.dmg instead.

For uninstallation just move Thonny to Bin.
