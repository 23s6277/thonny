This package will install Thonny VERSION — Python IDE for beginners.

More information about this release:
https://github.com/thonny/thonny/releases/tag/vVERSION

Some remarks about the installation procedure:

  • The recommended destination option is "For all users" (Thonny ends up under /Applications). If you don't have admin privileges then select "Install for me only" (Thonny ends up under ~/Applications).
  • Installer may seem to get stuck with "Running package scripts". Don't worry — it is compiling the Python standard library. Give it a minute or two.

More information about Thonny: https://thonny.org.
